# hubot-menu

Lunches near domaniewska

See [`src/menu.coffee`](src/menu.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-menu --save`

Then add **hubot-menu** to your `external-scripts.json`:

```json
[
  "hubot-menu"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
